Avatar

Avatar component is used to represent a user, and displays the profile picture and we can use initial letters of user in Avatar as well.

How to use ?

To use Avatar component in your projects you just need to include class avatar in your img tag.

Avatar Sizes

You can change the sizes of avatar by adding class avatar-sm, avatar-md or avatar-lg in the same img tag.
